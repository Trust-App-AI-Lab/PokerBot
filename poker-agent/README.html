Calculation Tools

Primary

equity.py

Monte Carlo equity计算。Hero手牌 vs 对手range，可选board。支持preset ranges（"20%"）和自定义range notation（"QQ+,AKs"）。10K sims误差±1-2%。
273 lines · Internal: range_parser.py
Primary

odds.py

Pot odds + EV计算。输入pot/call/equity，输出是否+EV。支持implied odds和SPR计算。MDF自动输出。
199 lines · Standalone
Primary

preflop.py

GTO开牌频率查询（6-max, 100BB）。输入手牌+位置，输出raise/fold频率。用于判断preflop是否在GTO range内。
447 lines · Standalone (internal chart)
Utility

evaluator.py

5-7张牌的牌力评估。输出牌型class（0=High Card → 9=Royal Flush）。主要用于showdown判断。
185 lines · Standalone
Internal

range_parser.py

Range notation解析器。"QQ+,AKs" → 具体combo列表。通常不直接调用——equity.py内部使用。
233 lines · Used by equity.py

Design Principles

Core

Teach How to Think, Not Rules

所有文档解释"为什么"而不是列"做什么"。表格嵌入解释性上下文中，不作为独立查找表存在。目标是让读者（bot或CoachBot）学会推理过程，而不是记忆规则。
Structure

Three-Layer Architecture

Thinking → Application → Tools。gto-fundamentals提供思维框架，四个应用文档处理具体场景，Python工具提供数字验证。每一层都能独立使用，但组合起来最强。
Integration

Cross-Reference, Not Duplication

概念只在一个地方深入解释，其他地方通过cross-reference指向它。例如position的完整理论在gto-fundamentals，preflop/postflop/sizing只引用不重复。
